รถเช่าอุดรธานี รุ่งโรจน์ คาร์เร้นท์ — LIFF Preview

ใช้หน้านี้เป็น URL สำหรับลงทะเบียน LIFF ใน LINE Developers (mobile-first preview).

ตัวอย่างการ์ดโปรไฟล์

ชื่อ: นางน้อย — ตำแหน่ง: แอดมินเพจ

Flex JSON (ตัวอย่าง): เปิด JSON

Webhook / API

Webhook endpoint (ตัวอย่างใน repo): /api/line/webhook

ถ้าต้องการทดสอบ webhook แบบ local ให้ใช้ `scripts/send-line-test.js` ใน repo

URL ที่จะใช้ลงทะเบียนกับ LINE

ใช้หนึ่งใน URL ด้านล่างเป็น “Endpoint URL / LIFF URL” ขึ้นกับการตั้งค่า:

  • GitHub Pages (ถ้าเปิดแล้ว): https://gittisak-go.github.io/th-carrent/apps/liff-preview/
  • Raw GitHub (ใช้ได้ทันที): https://raw.githubusercontent.com/gittisak-go/th-carrent/main/docs/apps/liff-preview/index.html

หมายเหตุ: ถ้า Pages ยังไม่แสดง (404) ให้รอ propagation หรือไปที่ Settings → Pages และตั้ง Source เป็น gh-pages (root) — ผมช่วยได้ถ้าคุณต้องการ

คำแนะนำด่วนหลังได้ URL

  1. ลงทะเบียน LIFF ใน LINE Developers โดยใส่ URL ข้างบน
  2. รับ LIFF ID แล้วใส่ไว้ในหน้า LIFF (ถ้ต้องการให้หน้าเรียก liff.init ให้ผมเพิ่มสคริปต์ให้)
  3. สำหรับ webhook ให้ตั้งค่า webhook URL เป็น URL ของเซิร์ฟเวอร์ที่รัน `api/line/webhook` (GitHub Pages ไม่รองรับ server-side webhook)
