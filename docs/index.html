th-carrent — หน้า LIFF ชั่วคราว

หน้านี้เป็นหน้า placeholder แบบ static เพื่อให้คุณมี URL สำหรับลงทะเบียน LIFF ได้ทันที หากต้องการใช้งาน LIFF จริง ให้ deploy โปรเจคไปที่ Vercel (แนะนำ) เพื่อให้มีทั้งหน้าเว็บและ webhook server

  • สิ่งที่หน้านี้ให้ได้: URL HTTPS สำหรับลงทะเบียน LIFF
  • ข้อจำกัด: เว็บนี้เป็น static page เท่านั้น — ไม่รองรับ webhook ของ LINE

เมื่อต้องการให้ผมช่วยต่อเพื่อดีพลอยจริงบน Vercel (รวม webhook) บอกผมได้ — ถ้าต้องการ URL ตอนนี้ ให้เปิด GitHub Pages ใน repo โดยเลือก source เป็น docs/.
